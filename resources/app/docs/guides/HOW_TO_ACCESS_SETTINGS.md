## Troubleshooting

### "I don't see the Settings button"

**On Web Browser:**
1. Hard refresh the page: `Ctrl + F5` (Windows) or `Cmd + Shift + R` (Mac)
2. Clear browser cache
3. Make sure you're on: https://danfosal-app.web.app

**On Desktop App:**
1. Close and restart the app
2. If Settings still doesn't appear, reinstall the app

**On Android:**
1. Try the **Quick Actions** menu (purple button bottom-right)
2. If not there, update the app:
   - Download latest APK from Firebase
   - Install over existing app
3. Or reinstall completely

### "Quick Actions button is missing"

The purple button should be at the bottom-right of your screen. If you don't see it:
1. Scroll up to the top of the page
2. Look for the **top-right gear icon (âš™ï¸)** instead

## Still Can't Find It?

If you've tried everything and still don't see the Settings option:

1. **Check the deployment:**
   - Web: https://danfosal-app.web.app
   - Make sure it shows 25 files deployed

2. **Verify the version:**
   - Open Developer Console (F12)
   - Check for any JavaScript errors

3. **Try another device:**
   - If desktop doesn't work, try mobile
   - If mobile doesn't work, try web browser

4. **Last resort:**
   - Uninstall the app completely
   - Download fresh from: https://danfosal-app.web.app
   - Install and open
