## How to Update Other Computers:

### Method 1: Copy the App Folder (Recommended)
1. Close the Danfosal app on all computers
2. Copy the entire folder:
   ```
   release-build\danfosal-app-win32-x64
   ```
3. Replace the old folder on each computer with this new one
4. Open the app - you're updated!

### Method 2: Use the Packager Script
1. Copy `package-for-work-pc.ps1` to the work PC
2. Run it to package the app
3. Transfer and install

## Verify the Update:
1. Open the Danfosal app
2. Go to **Store Sales**
3. Look for the purple **"Scan Store Invoice"** button
4. If you see it, the update is successful! ‚úÖ

## Testing the New Scanner:
